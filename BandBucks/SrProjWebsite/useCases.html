Use Cases

First Time User

  • Create account username/password
  • Edit user profile - contact info
  • Creates band
  • Edit band profile - band description, contact info, and band members
  • Adds bandmates by username
  • Selects Equipment/Debt tab
  • Inputs items owned personally and by the band
  • Selects Reports tab
  • Views band reports
  • Logs out

Repeat User

  • Login
  • Selects band from drop down menu
  • Selects Cash tab
  • Inputs latest expenditures ie. gas, food
  • Inputs latest revenues ie. t-shirts sold
  • Selects Equipment/Debt tab
  • Records sale of merchandise
  • Selects Reports tab
  • Views band reports
  • Logs out
